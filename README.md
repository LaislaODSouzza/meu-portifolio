# Meu Portfólio

Este projeto foi desenvolvido para centralizar minha trajetória acadêmica, projetos e formas de contato.

## Tecnologias Utilizadas
- **HTML5:** Estrutura semântica do conteúdo.
- **CSS3:** Estilização avançada com Grid Layout, Flexbox e Media Queries para responsividade.
- **JavaScript:** (Em implementação) para interatividade e lógica de interface.
  
## Funcionalidades
- **Layout Adaptável:** O site se reorganiza automaticamente para telas de celulares e tablets.
- **Seção de Tech Stack:** Exibição dinâmica de ícones das linguagens e ferramentas que utilizei.
- **Links Diretos:** Integração com GitHub, Beecrowd e contato via e-mail.
